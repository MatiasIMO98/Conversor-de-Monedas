<h2>Descripción del proyecto</h2>
Este proyecto corresponde a un Desafío de Alura Latam llamado "Practicando con Java: Challenge Conversor de Monedas". El objetivo de este desafío es Convertir el valor ingresado de una moneda a otra, esto mediante la utilización de una API conectada a ExchangeRate.
El sistema permite, mediante un menú por consola, seleccionar la moneda de cambio e introducir el valor a convertir.

<h2>Característica Principal</h2>
Mediante un menú desplegado a través de la consola, con 6 opciones disponibles, es posible convertir de una moneda a otra, principalmente de Dolares americanos a otras monedas, como lo son el Peso Argentino, el Real Brasileño y el Peso Chileno, y viceversa.
